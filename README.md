# 🏭 Manufacturing Equipment Output Prediction

This project is a Machine Learning application that predicts the number of parts produced per hour in a manufacturing process using machine parameters.

---

## 🚀 Project Overview

The goal of this project is to optimize production efficiency by predicting machine output based on operational parameters such as temperature, pressure, cycle time, and more.

---

## 📊 Features Used

- Injection Temperature
- Injection Pressure
- Cycle Time
- Cooling Time
- Material Viscosity
- Ambient Temperature
- Machine Age
- Operator Experience
- Maintenance Hours

---

## 🎯 Target

- Parts Produced Per Hour

---

## 🧠 Model Used

- Linear Regression

---

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
